## 技术栈

+ Express
+ Mongodb
+ Bootstrap
+ jQuery

## 用法
1. 克隆库到本地 `git clone git@github.com:jia58960/express-movie`
2. 安装所有依赖项: `npm install & bower install`
3. 修改 `app.js`中的MongoDB配置项，
4. 跑起来: `grunt`
5. 打开浏览器访问: `http://localhost:3000`
